## 💰 Expense Tracker Dashboard UI

## 📌 About
This project is a simple Expense Tracker Dashboard built using HTML and CSS.  
It simulates a financial dashboard interface where users can view their total balance, income, expenses, and transaction history.

---

## 🎯 Features
- Clean and modern dashboard UI  
- Summary cards for balance, income, and expenses  
- Transaction table for tracking records  
- Color-coded income and expense indicators  
- Responsive layout for different screen sizes  

---

## 🛠 Tech Stack
- HTML5  
- CSS3  

---


## 📸 Preview
A dashboard interface displaying financial summary cards and a transaction list, similar to real-world fintech applications.
